## Project setup

```bash
$ npm install
```

## Start dev Docker containers
```bash
docker-compose -f docker-compose.dev.yml up -d
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
